# 模板扩库候选清单 v1

## 目标
本清单用于扩大本地模板库覆盖面，优先保证常见物品都能命中“相似模板”，从而提升 `reuse / adapt` 命中率。

原则：
- 最终入库对象必须是 **体素模板**，不是 LEGO 砖块模板
- LEGO / MOC 站点只作为题材与轮廓灵感来源
- 优先采用可公开引用、适合体素化的 low-poly / voxel 参考
- 遇到授权或用途不合规的来源时，必须报备并跳过

## 优先扩仓范围
首轮按 3 类推进，共 18 个候选：
- 动物 7 个
- 车辆 6 个
- 建筑 5 个

## 合规状态说明
- `approved`
  可作为参考来源进入模板设计
- `inspiration_only`
  只可借鉴题材、轮廓和类别，不可直接照搬造型细节
- `blocked`
  不纳入本轮模板扩库，已报备并跳过
